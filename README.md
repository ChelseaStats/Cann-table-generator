### cann

> cann table generator from html table linked.

##### Requires

PHP server + skills
cUrl extension
access to a website with a well-formed html table you wish to process (bbc, sky, espn, thechels.co.uk etc).

##### Example

http://www.thechels.co.uk/premier-league-cann-table/
